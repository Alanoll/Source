Инициализация проекта с помощью Grunt.js

Инициализация проекта

Ининциализация проекта происходит с помощью Grunt.js. Вам необходимо выполнить следующие команды:

$ cd Source $ npm install $ grunt init

При этом все нобходимые файлы будут скопированы в соседнюю с Source папку Specs. Если вы хотите поменять папку, в которую проект будет инициализироваться, необходимо изменить значение поля pathToSpecs в файле настроек gruntOptions.js на необходимый вам путь к папке.

Обновление проекта

Для обновления ядра Source, например, после обновления до последней версии, необходимо использовать команду:

$ grunt update

Все файлы ядра будут минимизированы и копированы в папку проекта.

Деплой ядра на сервер через sftp

Для того, чтобы произвести деплой файлов на сервер, необходимо прописать соответствующие настройки доступа к серверу в файле gruntOptions.js, а так же пары имя/пароль в файле .ftppass. После заполнения этих настроек деплой запускается командой:

$ grunt deployServer

Все необходимые файлы будут минимизированы и залиты на сервер.
